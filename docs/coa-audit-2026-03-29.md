# COA Audit - March 29, 2026

## Scope
This audit separates storefront-level verification from supplier-level verification.

What is already verified from public competitor listings:
- Product naming used for mirrored catalog items
- Public retail pricing from Super Human Peptides shop listings
- Publicly listed format references where the storefront made them visible

What is not yet verified and should not be treated as production-accurate without supplier proof:
- Actual peptide identity
- Actual potency / assay content
- Actual purity
- Blend composition
- Sterility / endotoxin suitability for injectable-format items
- Residual solvent or contamination profile
- Lot-level batch consistency

## Production Accuracy Rule
A product should only be called production-accurate when all of the following are true:
1. The storefront name matches the supplier batch name.
2. The storefront format label matches the supplier COA and label artwork.
3. The storefront strength matches the supplier assay or content statement.
4. A lot-specific COA exists for the exact batch being sold.
5. The COA includes identity and purity evidence appropriate to the format.
6. The batch ID on the COA can be mapped to the storefront listing and fulfillment inventory.

## Required COA Fields
For every production SKU, require:
- Supplier name
- Product name
- Batch or lot number
- Manufacture date and test date
- Net content or stated strength
- Identity method listed explicitly
- Purity method listed explicitly
- Purity result
- Assay or content result
- Signature or release authorization

For injectable or vial-format products, also require when available:
- Sterility statement
- Endotoxin statement
- Reconstitution or storage guidance

For blends, also require:
- Each component named individually
- Each component amount stated individually
- Assay or content evidence that supports the combined label claim

For capsule products, also require:
- Count per bottle
- Per-capsule strength
- Excipients or inactive ingredients

## Recommended Release Workflow
1. Request the current supplier COA PDF for every live SKU.
2. Build a simple table with columns:
   - SKU
   - storefront format
   - supplier format
   - batch ID
   - purity
   - assay/content
   - verified by
   - approved for sale
3. Do not release a SKU as production-accurate until storefront format matches supplier batch proof exactly.
4. For blends and capsule products, verify composition details before publishing final label art.
5. Add lot-specific COA links to the COA library only after batch IDs are mapped to inventory.
